# Assets do LOGOS

Coloque aqui o arquivo oficial do logo principal do sistema.

Nomes reconhecidos automaticamente pela interface:
- logo_sistema.png
- logo_sistema.jpg
- logo_sistema.jpeg
- logo_sistema.webp
- logo.png
- logos.png

Opcionalmente, defina `SISTEMA_LOGO_PATH` no `.env` apontando para o arquivo desejado.

Exemplo:

SISTEMA_LOGO_PATH=assistente_financeiro/assets/logo_sistema.png
